#  Bot project from the School of Machines and Make Believe.
## Berlin, July 2019

## Instructions

* `install nodejs`
* `install npm`
* `npm install fs`

### To run 

* Go to folder on terminal
* `node bots.js`
* check out https://twitter.com/raid


## Text Generator 
 
* `npm install tracery-grammar --save`

## Image Scraper 

* `npm install request`
* `npm install images-scraper`
